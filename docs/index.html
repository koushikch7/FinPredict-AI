Architecture

How a prediction flows

From stock symbol to actionable insight — with technical analysis, sentiment, and AI reasoning at every step.

STEP 1

Fetch Market Data

Yahoo Finance v8 API provides OHLCV data, fundamentals, and historical prices — no API key required.

STEP 2

Compute Indicators

25+ technical indicators calculated: RSI, MACD, Bollinger, ADX, ATR, Stochastic, CCI, and more. Alpha158-inspired.

STEP 3

Analyze Sentiment

FinBERT processes recent news headlines. Scores aggregated into bullish/bearish/neutral with confidence.

STEP 4

AI Ensemble

Multiple AI providers analyze technicals + sentiment. Strategies: Buffett, Lynch, Graham, Momentum, MeanReversion, Balanced.

STEP 5

Conviction Score

Signals aggregated with weighted voting. Final BUY/SELL/HOLD with conviction %, price targets, and stop-loss.

⚙️ Stack: React 19 · TypeScript 5 · Vite · Tailwind 4 · Express 4 · SQLite (WAL mode) · Docker. Single container, one .env, port 3000. Background jobs handle daily predictions, market-hours refresh, and automated backups.

Quick start

One command to clone, one to configure, one to launch. Docker does the rest.

# 1. Clone the repository
git clone https://github.com/koushikch7/FinPredict-AI.git
cd FinPredict-AI

# 2. Configure environment
cp .env.example .env
nano .env  # Add your AI provider keys

# 3. Build and run
docker build -t finpredict-ai .
docker run -d --name finpredict \
  -p 3000:3000 \
  --env-file .env \
  -v finpredict-data:/app/data \
  finpredict-ai

# Open the dashboard
open http://localhost:3000
# Required: At least one AI provider
GEMINI_API_KEY=AIzaSy...your-key
# Or use Arbiter gateway for multi-provider routing
ARBITER_API_KEY=gwtk_your_arbiter_token
ARBITER_BASE_URL=https://your-arbiter-host/v1

# Optional: OpenAI for additional coverage
OPENAI_API_KEY=sk-...your-key

# Security (generate your own!)
JWT_SECRET=your-256-bit-secret
COOKIE_SECRET=your-256-bit-secret

# Optional: Zerodha broker integration
KITE_API_KEY=your-kite-api-key
KITE_API_SECRET=your-kite-api-secret

# Optional: FinBERT sentiment (HuggingFace)
HF_API_KEY=hf_...your-key
# Prerequisites: Node.js 20+, npm 10+

# 1. Install dependencies
npm install

# 2. Start development server (hot reload)
npm run dev

# 3. Build for production
npm run build

# 4. Start production server
npm start

# Server runs on http://localhost:3000
# Health check
curl http://localhost:3000/api/health
# → {"status":"ok","version":"1.6.4","uptime":12345}

# Register a user
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"demo","email":"demo@example.com","password":"SecurePass123!"}'

# Login and get token
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"demo@example.com","password":"SecurePass123!"}'

# Get predictions (with Bearer token)
curl http://localhost:3000/api/predictions/RELIANCE.NS \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

Investment Strategies

6 AI-powered strategies

Each strategy analyzes stocks through a different lens. Combine them for ensemble signals.

🏛️

Buffett

Warren Buffett's value investing. Quality companies with durable competitive advantages, strong moats, and reasonable valuations.

📈

Lynch

Peter Lynch's growth approach. Find "ten-baggers" — fast-growing companies with strong earnings momentum and market expansion.

📚

Graham

Benjamin Graham's deep value. Margin of safety focus — stocks trading below intrinsic value with strong balance sheets.

🚀

Momentum

Ride the trend. Identifies stocks with strong price momentum, high RSI, and increasing volume. Best for trending markets.

🔄

MeanReversion

Contrarian plays. Finds oversold stocks due for a bounce — RSI extremes, price below moving averages, sentiment lows.

⚖️

Balanced

Diversified approach. Combines signals from all strategies with weighted voting for robust, lower-variance predictions.
