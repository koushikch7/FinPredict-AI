## ⚡ Quick Start {#quick-start}

<div class="quickstart-box">
<h3>🐳 Docker (Recommended)</h3>

```bash
# Clone the repository
git clone https://github.com/koushikch7/FinPredict-AI.git
cd FinPredict-AI

# Configure environment
cp .env.example .env
nano .env  # Add your AI provider key (Gemini/OpenAI/Arbiter)

# Build and run
docker build -t finpredict-ai .
docker run -d --name finpredict \
  -p 3000:3000 \
  --env-file .env \
  -v finpredict-data:/app/data \
  finpredict-ai

# Access at http://localhost:3000
```

</div>

<div class="alert alert-info">
  <strong>💡 Tip:</strong> Only one AI provider key is required. 
  <a href="https://ai.google.dev">Get a free Gemini API key</a> to start.
</div>
