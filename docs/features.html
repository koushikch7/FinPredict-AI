📚 Features Guide

Comprehensive documentation of all FinPredict AI capabilities.

🤖 AI-Powered Predictions

🧠

Multi-Provider Ensemble

FinPredict AI uses multiple AI providers for robust predictions:

  • Google Gemini — Primary provider with excellent reasoning
  • OpenAI GPT-4 — Alternative provider for comparison
  • Arbiter Gateway — Unified endpoint with automatic failover to 11+ providers

The system generates predictions using 6 investment strategies:

  • Momentum Trend-following with RSI, MACD momentum
  • Value Undervalued stocks via P/E, P/B analysis
  • Growth Revenue/earnings growth focus
  • GARP Growth at Reasonable Price (PEG ratio)
  • Swing Short-term reversal opportunities
  • Contrarian Against-the-crowd plays
📊

Conviction Scoring

Each prediction includes a conviction score (0-100%) based on:

  • Technical indicator alignment (RSI, MACD, Bollinger bands)
  • Sentiment score from FinBERT analysis
  • Historical accuracy of similar signals
  • Market conditions (volatility, trend strength)

Score interpretation:

  • 80-100% — Strong conviction, multiple confirming factors
  • 60-79% — Moderate conviction, some conflicting signals
  • 40-59% — Weak conviction, consider waiting
  • Below 40% — No clear signal, hold or avoid

🧠 FinBERT Sentiment Analysis

📰

NLP-Powered News Analysis

FinPredict AI uses HuggingFace's FinBERT model — a BERT variant fine-tuned on financial text:

  • Real-time processing of news headlines
  • Three-way classification: Bullish, Bearish, Neutral
  • Confidence scores for each classification
  • Aggregated sentiment over configurable time windows

Sentiment is integrated into AI predictions as a key input factor.
