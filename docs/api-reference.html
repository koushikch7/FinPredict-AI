Complete REST API documentation. All endpoints require Authorization: Bearer <token> unless marked public.

🔐 Authentication

MethodEndpointAuthDescription
POST/api/auth/registerPublicRegister new user
POST/api/auth/loginPublicLogin and receive JWT
POST/api/auth/logoutRequiredLogout and clear session
GET/api/auth/meRequiredGet current user profile
PATCH/api/auth/meRequiredUpdate user settings
POST/api/auth/change-passwordRequiredChange password
